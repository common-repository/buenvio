== Descripción ==

Utiliza este plugin gratuito para integrar los envíos con Buenvio a tu tienda woocommerce. Ya no sigas estimando costos de envió y gestionando el llenado de datos de envíos e impresión de label manualmente.

Los precios se calculan automáticamente por buenvio, logrando así dar mejores precios a tus clientes así como evitar posibles errores de traspaso manual de información.

=== Este plugin solo funciona para envíos en República Dominicana ===
